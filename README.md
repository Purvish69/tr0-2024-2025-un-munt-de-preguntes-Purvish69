# Transversals

## Autoescuela - Proyecto de Gestión de Autoescuela

### 👥 Integrantes
- Purvish Patel

### 📌 Descripción del proyecto
Este proyecto consiste en un formulario tipo test para el permiso de conducir B de coches. Permite responder a una serie de preguntas tipo examen, simulando el test teórico de la autoescuela.

Este fue mi primer proyecto realizado durante el curso, con el objetivo de aprender los fundamentos del desarrollo web, como la creación de formularios, manejo de respuestas y lógica básica con JavaScript.
