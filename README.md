## Autoescuela - Proyecto de Gestión de Autoescuela

### 👥 Integrantes
- Purvish Patel

### 📌 Descripción del proyecto
Este proyecto es un sistema de gestión para una autoescuela de coches. Permite organizar y administrar las principales funciones de una autoescuela, como el registro de alumnos, gestión de clases teóricas y prácticas, seguimiento de progreso y control general de la formación de los estudiantes.

Este fue mi primer proyecto realizado durante el curso, con el objetivo de aprender los fundamentos del desarrollo web y la organización de aplicaciones completas.
